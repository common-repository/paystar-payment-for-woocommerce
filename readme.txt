برای درخواست پذیرندگی و دریافت اطلاعات ترمینال از لینک زیر اقدام نمایید :
https://my.paystar.ir/

== Description ==

with this plugin, you can easily add paystar.ir gateway to woocommerce checkout system
با این پلاگین میتونید درگاه پرداخت پی استار رو به سیستم تسویه حساب و پرداخت ووکامرس اضافه کنید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی ووکامرس ، زیرمنوی تنظیمات ، به سربرگ تسویه حساب رفته و روی اسم درگاه پی استار کلیک کنید
